Phase 4 DE Overview

Atlas-level contrast summary template

Scope

State the active context, annotation layer, DE round, contrast key, and selected contrast. Explain the biological meaning of the contrast and whether this run uses a simple factor, a composite condition, a within-group contrast, or an interaction.

Run Settings

Summarize the DE settings that materially affect interpretation, including engine, available evidence sources, explicit contrasts, minimum count thresholds, and whether pseudobulk was run.

Atlas-Level DE Summary

Give a concise summary of how strong and how distributed the signal appears across the atlas. State whether the contrast looks focal, diffuse, weak, or compartment-specific.

Coverage And Skip Summary

Cluster Status Evidence Sources Main Note
C00 Tested or skipped Cell-level only, pseudobulk only, or both One short sentence on power, skip reason, or confidence level.

Strongest Responding Clusters

Highlight only the clusters with the most interpretable signal. Prefer those with coherent biology and replicate-aware support when pseudobulk is available.

Weak Or Ambiguous Clusters

Summarize clusters that look underpowered, inflation-prone, discordant across sources, or otherwise not yet convincing.

Pathway And Regulator Overview

Summarize broad pathway and regulator shifts across the atlas. Explain how DoRothEA, MSigDB, and PROGENy agree with or complicate the gene-level story.

Priority Shortlist For Phase 5

  • Name the first cluster that deserves a DE deep dive and why.
  • Name the second cluster if warranted.
  • Note any cluster that looks tempting but is still too weak or too underpowered.

Practical Takeaway

End with a short paragraph stating whether the contrast produced a useful atlas-level story and where the most trustworthy DE signal sits.
